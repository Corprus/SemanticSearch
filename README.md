# SemanticSearch

## Name
SemanticSearch

## Description
Семантический поиск по пользовательским документам с использованием моделей ML.
Пользователь может загружать документы, ьудет (за кредиты) производиться их индексация, и искать с использованием семантики по ним (за поиск тоже будут сниматься кредиты).
Цель: удобный семантический поисковик по загруженным текстовым документам пользователя
